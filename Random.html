There are two ways : 
1. Using Math.random()
2. Using Random class in Java


1. Using Math.random() :
   

    double random = Math.random() * Max + Min;
    or
    int random = (int )(Math.random() * Max + Min);
    

    This will give you value from Max to Min using Math.random()
    

    punch line:
    random() method returns a random number between 0.0 and 0.999
    

    

2. Using Random class in Java :
 

 Random rand = new Random();
